## Feature 5
This feature allows the user to disqualify a specific driver of a specific constructor in a specific race.
### Interface Description
The interface will consist of 3 boxes that the user can submit a race (rID), representing the race they would like to query, a driver (dID), representing the driver they would like to disqualify, and a constructor (cID), representing the constructor of the driver they would like to disqualify. There will also be a button that the user can click to submit their query.
### Users
The users of the interface will include motorsport enthusiasts, analysts, and bettors who want to disqualify a driver from a specific race. Users might be team strategists aiming to review race conditions, journalists focused on race events, or fantasy league organizers updating contest outcomes.

The user will interact with the interface by entering their desired parameters into the boxes, then clicking the button, and the interface will process the disqualification request.
